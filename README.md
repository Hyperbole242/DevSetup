DevSetup 🚀
Bienvenue dans mon premier environnement de développement !
Ce projet a pour but de poser les bases de mon parcours en développement web fullstack. Ici, je rassemble mes outils, mes notes et mes premières expérimentations avec Git et GitHub.

🎯 Objectifs
Installer et configurer les outils essentiels (VS Code, Git, Node.js).

Créer mon premier repository GitHub et comprendre le workflow de base.

Prendre en main le terminal et les commandes Git fondamentales.

📂 Structure du projet
README.md → ce document, qui explique le projet.

notes/ → un dossier contenant mes notes de la semaine (3 fichiers markdown).

.gitignore → pour garder le repo propre et éviter d’y pousser des fichiers inutiles.

⚡ Premiers pas
Clonez le repo :

bash
git clone https://github.com/<mon-username>/DevSetup.git
Explorez le dossier notes pour suivre mon apprentissage.

Lisez le README pour comprendre la logique du projet.

🌱 Pourquoi ce projet ?
Parce que chaque développeur commence par mettre en place son environnement. Ce repo est mon carnet de bord, une trace de mes premiers pas dans le monde du code.

✨ Bonus
Configuration de Nodemon pour automatiser le redémarrage.

Ajout d’un .editorconfig pour garder un style de code cohérent.

Création d’une branche dev pour tester avant de merger dans main.
